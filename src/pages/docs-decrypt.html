How to Decrypt Vulnerability Reports

When you receive an encrypted vulnerability report via email, follow these steps to decrypt it locally.

  1. 1

    Download the Encrypted Package

    BLT-Zero will send you an email with a JSON attachment. Download the blt-zero-*.json file to your local machine.

  2. 2

    Ensure You Have Your Private Key

    You should have generated a private key during onboarding. The file is typically named private_key.jwk.

    Security Tip: Keep your private key secure and never share it. It's the only way to decrypt reports.

  3. 3

    Run the Decryption Script

    Navigate to your BLT-Zero repository and run: 

    python tools/org_decrypt.py private_key.jwk package.json

    Replace package.json with the path to your downloaded encrypted file.

  4. 4

    Review the Decrypted Report

    The script will output a report.json file containing:

    • • Vulnerability description
    • • Target URL and domain
    • • Detailed markdown report (if provided)
    • • Base64-encoded screenshots
    • • Reporter's BLT username (if provided)

Prerequisites

  • ✓ Python 3.7 or higher
  • ✓ Required packages: cryptography, jwcrypto
  • ✓ Install via: pip install -r tools/requirements.txt
