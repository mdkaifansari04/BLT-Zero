Organization Onboarding

What Organization Admin Does

  1. 1. Generate keypair locally using the provided script (private key stays with your organization)
  2. 2. Upload public key to BLT-Zero via the admin onboarding page
  3. 3. Decrypt incoming vulnerability reports locally with your private key

Important: Your private key never leaves your infrastructure. Only you can decrypt the vulnerability reports.

Available Scripts

We provide Python scripts in the BLT-Zero repository under tools/ directory.

1. Generate Keypair

python tools/org_keygen.py

Generates private_key.jwk and displays public key for upload.

2. Decrypt Reports

python tools/org_decrypt.py private_key.jwk package.json

Decrypts the encrypted package and outputs report.json.
